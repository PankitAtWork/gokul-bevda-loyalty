# Arduino Project

This is a separate Arduino project located within the workspace.

## Getting Started

### Prerequisites

1.  **Arduino IDE**: Download and install from [arduino.cc](https://www.arduino.cc/en/software).
2.  **Hardware**: An Arduino board (Uno, Nano, Mega, etc.) and a USB cable.

### How to Run

1.  Open the `arduino_project.ino` file in the Arduino IDE.
2.  Select your board:
    *   Go to **Tools** > **Board** and select your specific Arduino board.
3.  Select the port:
    *   Go to **Tools** > **Port** and select the port your board is connected to.
4.  **Verify/Compile**: Click the checkmark icon (or generic "Verify" button) to check for errors.
5.  **Upload**: Click the right-arrow icon (or "Upload" button) to flash the code to the board.

## Functionality

Currently, this project implements a basic **Blink** example:
*   Blinks the built-in LED on and off every 1 second.
